# GitHub Spark prompt — BlackBugsAI UI

Design a futuristic cyber-neon interface for **BlackBugsAI** with both:
1. Android APK dashboard UI (Flutter-friendly components)
2. Web admin panel UI (responsive desktop-first)

Requirements:
- Color system: dark background, cyan/purple accents, clear success/warn/error states.
- Screens: Login, Server setup (GCP host/token), Dashboard, Agents, Tasks, Docker controls, Terminal, Settings.
- Include reusable components: stat cards, neon buttons, command input, task timeline, agent status chips.
- Optimize for accessibility: high contrast, readable typography, large tap targets.
- Export assets/specs that can be implemented in Flutter and plain HTML/CSS.
