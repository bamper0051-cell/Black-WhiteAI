Всё что нужно —
уже внутри

Никаких сборок и интеграций. Одна команда и бот работает со всем функционалом.

🧠
Автономный ИИ-агент
Работает в цикле: LLM → инструменты → анализ результата → следующий шаг. До 5 раундов без участия человека. Может писать и запускать код сам.
🔧
12 встроенных инструментов
TTS, генерация видео/изображений, sandbox для кода, создание новых ботов, веб-поиск, скачивание страниц, управление процессами.
🎙
TTS + Видео
edge-tts бесплатно или ElevenLabs. Сборка видео из картинок + аудио через ffmpeg. Скачивание YouTube в MP3/MP4.
🔑
Авторизация с капчей
Математическая капча на входе, регистрация/вход в аккаунт, bcrypt-хеши паролей, сессии, система привилегий (user/vip/admin/owner).
👑
Панель администратора
Управление пользователями, рассылки, shell-команды, просмотр процессов, изменение .env в реальном времени, история команд.
⚡
45+ LLM провайдеров
Groq, Gemini, OpenRouter, Cerebras, SambaNova, DeepSeek, Mistral, OpenAI, Claude и ещё 36. Переключение в 1 клик, авто-конфиг.
🎨
Генерация изображений
Pollinations (бесплатно), DALL-E 3, Stability AI, HuggingFace. Авто-выбор лучшего провайдера.
🤖
Самоулучшение
Агент пишет новые модули, устанавливает зависимости и даёт инструкцию по интеграции. Создаёт новых ботов по описанию.
🐳
Docker-ready
Dockerfile и requirements.txt включены. Один docker run и бот работает в изоляции с автоперезапуском.
