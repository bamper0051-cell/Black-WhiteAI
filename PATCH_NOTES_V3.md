# AGENT_SMITH — Patch Notes v3 (синтаксис-фикс)

## Изменённый файл: agent_session.py

### Проблема
АГЕНТ_0051 застревал в бесконечном цикле фикса синтаксиса:
- `unterminated triple-quoted string literal` — LLM генерировал незакрытые docstring
- После 3 попыток: `пересоздаю код`, но пересозданный код тоже ломался
- `line 1: invalid syntax` — LLM возвращал объяснение вместо кода, весь текст шёл как "код"

### Исправления

#### 1. `_sanitize_code` — агрессивная очистка (было: только Unicode замены)
- BOM удаление
- Замена ведущих табов → 4 пробела
- Автоматическое закрытие незакрытых тройных кавычек (определяет нечётное число `"""` или `'''`)

#### 2. `_smart_syntax_fix(code, error_msg, error_line)` — NEW
Детерминированный фикс без LLM. Вызывается ПЕРВЫМ, экономит LLM-попытки:
- `unterminated triple-quoted string` → добавляет `"""` или `'''` в конец
- `unterminated string literal` → закрывает строку на конкретной строке
- `invalid syntax` на строке 1 → обрезает мусор/объяснение перед кодом
- `expected an indented block` → вставляет `pass`
- `invalid character` → убирает все не-ASCII

#### 3. `_extract_code_robust(raw)` — NEW
Надёжное извлечение кода из LLM-ответа:
- ````python ... ` `` (стандарт)
- ` `` ... ` `` (без языка)
- Код с первой Python-строки (если нет блока)
- Fallback placeholder если ничего не найдено

#### 4. Цикл lint/fix — улучшен
```
Было:  3 попытки LLM → пересоздание → ещё 3 LLM → ...

Стало: 1. _smart_syntax_fix() [детерминированно, мгновенно]
       2. Если не помог → LLM фикс (до 5 попыток)
       3. Если та же ошибка повторяется ≥2 раз → пересоздание
       4. MAX_REGEN=2: максимум 2 полных пересоздания
       5. После MAX_REGEN → сохраняем что есть, не зависаем
```

#### 5. `_generate_code_with_smith` — улучшен промпт
- Явный запрет Unicode в промпте
- Использует `_extract_code_robust` вместо голого regex
- Fallback placeholder если LLM вернул пустой код

#### 6. `_fix_code_with_smith` — улучшен
- `_extract_code_robust` вместо голого regex
- `_sanitize_code` применяется сразу после LLM
- Если LLM вернул текст без кода → возвращаем оригинал (не ломаем)
