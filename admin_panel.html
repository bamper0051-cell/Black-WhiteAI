☁️ Google Cloud Engine — инструкция запуска

      

        

          1. Создай VM (бесплатный tier e2-micro):

          gcloud compute instances create blackbugsai \
  --machine-type=e2-medium --zone=us-central1-a \
  --image-family=ubuntu-2204-lts --image-project=ubuntu-os-cloud \
  --boot-disk-size=30GB --tags=http-server,https-server

          2. Открой порты (firewall):

          gcloud compute firewall-rules create allow-blackbugs \
  --allow tcp:8080,tcp:5000 --source-ranges=0.0.0.0/0

          3. SSH → установи Docker:

          curl -fsSL https://get.docker.com | sh
sudo usermod -aG docker $USER && newgrp docker

          4. Загрузи проект и запусти:

          git clone https://github.com/YOUR/repo /app/BlackBugsAI
cd /app/BlackBugsAI && cp .env.example .env
nano .env  # заполни BOT_TOKEN, ADMIN_WEB_TOKEN и др.
docker compose up -d

          5. Открой панель:

          http://EXTERNAL_IP:8080
          
⚠️ Для HTTPS — используй Cloudflare Tunnel (бесплатно) или nginx + certbot
