# BIG AUTOPATCH

## Что изменено

### 1. Капча и регистрация
- Капча заменена с перегруженного "киберпанка" на более простые задачи:
  - пример на сложение/вычитание
  - подсчёт эмодзи
  - слово наоборот
  - продолжение числовой последовательности
- Тексты регистрации переписаны в спокойный пошаговый вид:
  - Шаг 1 из 3 — проверка
  - Шаг 2 из 3 — никнейм
  - Шаг 3 из 3 — PIN
- Кнопки обновлены: вместо странных формулировок теперь "Обновить" и "Подсказка".

### 2. Admin panel
- Панель больше не привязана жёстко к `http://localhost:8080`.
- Если панель открыта через туннель или по внешнему домену, она использует `window.location.origin`.
- Поддержан автоподхват токена из URL:
  - `/panel?token=...`
  - `/panel?base=https://...&token=...`
- Улучшена ошибка подключения: 401 теперь показывается как проблема токена, а не как абстрактный HTTP fail.

### 3. Mobile / APK server side
- `admin_web.py` теперь принимает токен не только через `X-Admin-Token`, но и через:
  - `Authorization: Bearer <token>`
  - `X-Api-Key`
- Расширен CORS для мобильных клиентов.
- Добавлен `/api/mobile/bootstrap` для быстрой серверной проверки с телефона.

## Чего в этом пакете НЕТ
- В архиве не найден исходный Android / APK проект.
- Поэтому исправлена серверная часть и web panel, но сам нативный APK-клиент в этом архиве не патчился.
